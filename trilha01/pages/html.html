O que é HTML?

HTML é uma linguagem de marcação que define o significado e a estrutura do conteúdo da web. Sua sigla significa HyperText Markup Language.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou de sites externos.

A HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais, como

<html>, <head>, <meta>, <link>, <title>, <body>, <header>, <div>, <h1>, <p>, <img>, <ul>, <ol>, <li>, <nav>, <main>, <button>, <footer>, <br>

Anatomia de um elemento HTML

Anatomia de um documento HTML

Vamos observar como os elementos individuais se combinam para formar uma página HTML completa.

Elementos HTML

<html>

Representa a raiz de um documento HTML.

<head>

Contém metadados do documento.

<meta>

Define metadados da página.

<link>

Define relação com recursos externos.

<title>

Define o título do documento, mostrado na aba do navegador.

<body>

Representa todo o conteúdo visível da página HTML.

<header>

Representa um cabeçalho da página ou de uma seção.

<div>

Container genérico usado para agrupar elementos.

<h1>

Representa o título principal de uma seção.

<p>

Representa um parágrafo de texto.

<img>

Usado para inserir imagens em uma página HTML.

<ul>

Representa uma lista não ordenada (com marcadores).

<ol>

Representa uma lista ordenada com numeração.

<li>

Define um item dentro de uma lista.

<nav>

Representa uma área de navegação com links.

<main>

Define o conteúdo principal da página.

<button>

Representa um botão clicável.

<footer>

Representa o rodapé da página ou de uma seção.

<br>

Cria uma quebra de linha no texto.
