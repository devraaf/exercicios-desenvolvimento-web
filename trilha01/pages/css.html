O que é CSS?

Cascading Style Sheets: folhas de estilo em cascata. Linguagem de design gráfico usada junto ao HTML para aplicar estilos como cores, margens, formas e fontes.

Anatomia CSS

Toda essa estrutura é chamada de conjunto de regras.

  • Seletor: seleciona o elemento HTML que será estilizado.
  • Declaração: regra como color: red;.
  • Propriedade: característica que será alterada (ex: color).
  • Valor: define a aparência da propriedade.

Cada conjunto de regras utiliza chaves { }. As propriedades são separadas por ; e os valores por :.
